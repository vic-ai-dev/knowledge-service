# 偿二代合规指南

## 1. 概述

本文档定义保险公司在《保险公司偿付能力管理规定》(偿二代二期)下的技术合规要求，覆盖偿付能力数据采集、计算、报告及系统对接。适用于数据平台、核心系统和精算系统的开发和运维人员。

## 2. 偿二代监管框架

### 2.1 三大支柱

| 支柱 | 内容 | 技术影响 |
|------|------|---------|
| 第一支柱 定量要求 | 量化资本要求、实际资本、最低资本计算 | 精算模型系统 + 数据平台 |
| 第二支柱 定性要求 | 风险管理要求与评估(SARMRA) | 风险管理系统、内部控制 |
| 第三支柱 市场约束 | 信息披露要求 | 报告系统、数据披露 |

### 2.2 偿付能力指标

| 指标 | 定义 | 监管要求 |
|------|------|---------|
| 综合偿付能力充足率 | 实际资本 / 最低资本 | ≥ 100% |
| 核心偿付能力充足率 | 核心资本 / 最低资本 | ≥ 50% |
| 风险综合评级 | 综合风险评价 | A/B/C/D 四级 |

## 3. 技术系统要求

### 3.1 数据采集与校验

| 数据源 | 数据内容 | 采集频率 | 数据质量要求 |
|--------|---------|---------|-------------|
| 承保系统 | 保单明细、保费收入 | T+1 | 准确率100%，完整率≥99.9% |
| 理赔系统 | 已决/未决赔款 | T+1 | 准确率100% |
| 财务系统 | 资产、负债、利润 | T+1 | 与总账一致 |
| 精算系统 | 准备金评估结果 | 月 | 精算报告校验 |
| 投资系统 | 资产配置、投资收益 | T+1 | 与托管行对账一致 |

### 3.2 数据质量要求

| 质量维度 | 要求 | 监控方式 |
|---------|------|---------|
| 完整性 | 关键字段非空率 ≥ 99.9% | 数据质量监控平台 |
| 准确性 | 需与源系统完全一致 | 交叉对账规则 |
| 及时性 | T+1 10:00前完成采集 | 数据时效监控 |
| 一致性 | 跨系统相同指标口径一致 | 指标一致性校验 |
| 可追溯性 | 报送数据可追溯到原始单据 | 数据血缘追踪 |
| 稳定性 | 同一指标不同期次不可出现大幅异常波动 | 波动监控规则 |

### 3.3 偿付能力计算流程

```
数据采集(各系统) → 数据校验 → 准备金评估 → 实际资本计算 → 最低资本计算 → 充足率计算 → 报告生成 → 报送
```

各环节自动化率目标 ≥ 90%，异常数据自动告警。

## 4. 报告报送

| 报告类型 | 频率 | 报送时限 | 报送系统 |
|---------|------|---------|---------|
| 偿付能力季度报告 | 季度 | 季后30天 | 监管报送系统 |
| 偿付能力年度报告 | 年度 | 次年4月30日 | 监管报送系统 |
| 偿付能力临时报告 | 触发时 | 知悉后3日内 | 监管报送系统 |
| 风险综合评级报告 | 季度 | 季后30天 | 监管报送系统 |
| SARMRA评估报告 | 年度 | 次年6月 | 监管报送系统 |

## 5. 系统对接架构

```
┌──────────┐   ┌──────────┐   ┌──────────┐
│ 业务系统 │ → │ 数据平台 │ → │ 偿付能力  │
│ 承保/理赔 │   │ ODS→DWD  │   │ 计算引擎  │
│ 财务/投资 │   │ 数据质量 │   │ 精算模型  │
└──────────┘   └──────────┘   └─────┬────┘
                                     ↓
┌──────────┐   ┌──────────┐   ┌──────────┐
│ 监管报送  │ ← │ 报告生成 │ ← │ 结果校验  │
│ 系统对接  │   │ 报表平台  │   │ 逻辑校验  │
└──────────┘   └──────────┘   └──────────┘
```

## 6. IT 合规检查清单

| 检查项 | 要求 | 周期 |
|-------|------|------|
| 偿付能力源数据完整性 | 核心字段无缺失 | 每日 |
| 数据对账一致性 | 财务账 vs 业务系统数据差异=0 | 每日 |
| 报送数据备份 | 每次报送文件加密归档 | 每季度 |
| 计算逻辑变更审批 | 精算模型变更需审批+版本管理 | 每次变更 |
| 数据权限审计 | 偿付能力数据访问权限季度审计 | 每季度 |
