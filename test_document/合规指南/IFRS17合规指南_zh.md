# IFRS17合规指南

## 1. 概述

本文档定义国际财务报告准则 IFRS 17《保险合同》在保险技术系统中的合规实施要求，覆盖数据采集、计量模型、分摊计算和报告披露等环节。适用于数据平台、精算系统和财务系统的开发和运维人员。

## 2. IFRS17 核心概念

### 2.1 核心变化
IFRS 17 取代 IFRS 4，主要变化包括：

| 方面 | IFRS 4 (旧) | IFRS 17 (新) |
|------|-------------|-------------|
| 收入确认 | 已收保费 | 基于保险服务提供确认 |
| 利润确认 | 首日利润可全部确认 | 需随服务逐步释放(CSM) |
| 折现率 | 不强制要求折现 | 需反映货币时间价值 |
| 假设更新 | 不强制锁定 | 需定期更新假设 |
| 披露 | 有限 | 大幅增加 |

### 2.2 关键指标

| 术语 | 缩写 | 说明 |
|------|------|------|
| 合同服务边际 | CSM | 未赚利润，随服务提供释放 |
| 保险收入 | Insurance Revenue | 基于已提供保险服务确认的收入 |
| 保险服务费用 | Insurance Service Expenses | 已发生赔付、费用的总和 |
| 保险财务收益/费用 | Insurance Finance Income/Expenses | 折现率变动带来的财务影响 |
| 亏损合同 | Onerous Contract | 预期赔付>收入的合同 |
| 履约现金流 | Fulfillment Cash Flows | 未来现金流现值 + 风险调整 |

## 3. 计量模型

IFRS17 提供三种计量模型：

| 模型 | 适用条件 | 主要特点 |
|------|---------|---------|
| 通用模型 (GMM/BBA) | 默认模型 | 逐期释放 CSM，假设需定期更新 |
| 保费分配法 (PAA) | 保险期 ≤ 1年 | 简化处理，近似保费收入 |
| 可变费用法 (VFA) | 具有直接分红特征的合同 | CSM 调整反映投资波动 |

## 4. 技术系统要求

### 4.1 数据要求

IFRS17 需要比 IFRS4 更细粒度、更多维度的数据：

| 数据维度 | 说明 | 来源系统 |
|---------|------|---------|
| 保单级数据 | 每份保单的现金流预测 | 承保/保单系统 |
| 分组信息 | 按年度/险种分组 | 分组规则引擎 |
| 精算假设 | 死亡率、退保率、费用率 | 精算系统 |
| 折现率曲线 | 不同期限的折现率 | 投资系统/市场数据 |
| 实际经验数据 | 实际赔付、退保、费用 | 理赔/保单/财务系统 |
| 合同修改 | 保单变更记录 | 保全系统 |

### 4.2 计算流程

```
数据采集 → 保单分组 → 现金流预测 → 折现 → 风险调整 → CSM计算 → 损益确认 → 报告生成
```

| 计算步骤 | 频率 | 说明 |
|---------|------|------|
| 初始确认 | 保单签发时 | 首次确认履约现金流 |
| 后续计量 | 每季度 | 更新假设、释放 CSM |
| 年度组关闭 | 年度 | 锁定分组，不再新增 |
| 亏损识别 | 每季度 | 判断是否形成亏损合同 |

### 4.3 系统架构建议

```
┌──────────┐   ┌─────────────┐   ┌──────────┐
│ 源数据系统 │ → │ IFRS17计算引擎 │ → │ 财务总账  │
│ 承保/理赔  │   │ 精算模型+CSM  │   │ GL系统   │
└──────────┘   └──────┬──────┘   └──────────┘
                       ↓
                ┌─────────────┐
                │ 披露报告生成  │
                │ 附注+明细表   │
                └─────────────┘
```

## 5. 披露要求

IFRS17 要求在财务报表中披露：

| 披露内容 | 技术实现 |
|---------|---------|
| 保险合同余额调节表 | 期初→本期变化→期末 |
| 保险收入分析 | 按险种/渠道/地区维度的收入分解 |
| CSM 调节表 | CSM 期初余额→新增→利息→释放→经验调整→期末 |
| 亏损合同明细 | 亏损合同的初始确认和后续计量 |
| 折现率假设 | 各期限折现率曲线 |
| 风险调整 | 风险调整方法、置信水平 |
| 理赔发展三角 | 历史年度赔款的发展趋势 |
